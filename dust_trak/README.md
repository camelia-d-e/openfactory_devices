## Adapteur DustTrak 
Cet adapteur peut fonctionner de deux façons: mode openfactory ou mode csv. Cela change la façon dont sont exportées les données du capteur. Il est aussi possible de changer la fréquence de lecture des données dans le même [fichier de configuration](config.json).

### Prérequis
Pour que l'initialisation de l'adapteur fonctionne, il faut s'assurer d'avoir le shortcut de l'application Environmental DustTrak Checkout sur son bureau et de changer l'image dans template si elle ne correspond pas à l'[image déjà présente](templates/dust_trak_shortcut.png) ci-dessous.<br>
![alt text](dust_trak_shortcut.png)
<br>
C'est le même principe pour l'image du logo de l'application dans la task bar, qui permet d'éviter de lancer plusieurs instances de l'application en même temps, si l'interface du bureau est différente de cette [image](templates/dust_trak_task_bar.png) ci-dessous, il faut reprendre une capture d'écran pour s'assurer que le tout va fonctionner.<br>
![alt text](dust_trak_task_bar.png)
